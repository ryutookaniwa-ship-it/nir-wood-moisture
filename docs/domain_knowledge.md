# NIR木材含水率予測 — ドメイン知識

> 出典: hirokenn/spectral_analysis プロジェクトの知見を統合・整理したもの

## 1. 課題の本質

- 入力: 木材表面の近赤外スペクトル (10000〜4000 cm-1, 1555波数点)
- 出力: 含水率 (%)
- 評価: RMSE

スペクトルには3種の情報が混在:
- **化学情報**: 水や木材成分由来の吸収
- **物理情報**: 散乱、表面状態、繊維方向、密度差
- **測定系の影響**: プローブ距離、光学系、器差

→ 前処理と検証設計の良し悪しが精度に直結

## 2. 近赤外スペクトルの基礎

### 何を測っているか
- 分子振動の倍音・結合音を観測
- 中赤外のような鋭いピークではなく、広く重なった吸収帯
- O-H関連帯域 (約7000 cm-1、約5200 cm-1) が水分情報に寄与

### 木材試料特有の難しさ
- **異方性**: 板目・柾目・追い柾で光の散乱挙動が変化
- **密度差**: 光路長や散乱強度に影響
- **乾燥収縮**: 測定中にプローブ-試料距離が変化する可能性
- **表面状態**: 粗さや局所構造が反射強度を変える

→ 「化学濃度が同じでもスペクトル形状が変わる」原因

## 3. ケモメトリクスの考え方

重視すべき点:
- 高次元・多重共線性のあるスペクトルを安定に扱う
- 前処理で不要変動 (散乱・ベースライン) を抑える
- 少数サンプルでも過学習しにくいモデルを選ぶ
- 未知樹種への外挿に近い状況 → 「汎化性」重視

## 4. 前処理手法

### 散乱補正
| 手法 | 概要 | 本コンペでの実績 |
|------|------|------------------|
| SNV | 各サンプルを平均0・分散1に正規化 | LOSO 15.99 (L2実験) |
| **MSC** | 参照スペクトルへの回帰でスケール差・オフセット差を補正 | **LOSO 15.73 (I2)** ← 確定最良 |
| Raw (補正なし) | — | LOSO 22.77 (大幅悪化) |

### 平滑化と微分
| 手法 | 概要 | 本コンペでの実績 |
|------|------|------------------|
| SG 1次微分 | ベースライン影響低減、ピーク変化強調 | w=9,p=2が最適 (J4で確定) |
| SG 2次微分 | 重なりピーク分離。ノイズ増幅に注意 | 未検証 |
| 窓幅 | 小窓 (w=5〜9) が有利 | w=9 > w=7 > w=11 (S実験) |

### EPO (External Parameter Orthogonalization)
- **本コンペ最大のブレイクスルー**: 樹種依存のスペクトル変動を直交除去
- n_components=5が最適 (n=6で+0.019、n=7で+2.37と急激悪化)
- bin_width=10が最適
- LB改善: 18.4 → 16.1 (B2実験)

### ターゲット変換
- y^0.27 (べき乗変換) が最適: 高含水率域の圧縮が有効
- p=0.270が真の最適 (p=0.265で+0.005、p=0.278で+0.015と急激悪化)
- LB改善: 16.1 → 15.395 (P1実験)

## 5. モデル選択

### 本コンペの結論

| モデル | 結果 | 理由 |
|--------|------|------|
| **LightGBM** | ✅ 最良 | ff=0.07のrandom subspaceが樹種汎化に効果的 |
| XGBoost | ❌ 劣る | level-wise成長がleaf-wiseより不利 (K3) |
| SVR (RBF) | ❌ 劣る | EPO後もLGBMに及ばない (L5) |
| Random Forest | ❌ 劣る | バギングはEPO後でも樹種汎化に不利 (F2) |
| MLP | ❌ 壊滅 | LOSO良いほどLB悪い逆相関 (F1, H1) |
| 1D-CNN | ❌ 大幅悪化 | N=1322では過学習 (X) |
| Transformer | ❌ 悪化 | gap=+3.13 (F3) |

**重要知見**: ニューラルネットはすべて失敗。Tree + random subspaceが小サンプルスペクトルデータの本質

### LightGBMの最適パラメータ
- leaves=63, min_child_samples=10 (I2-params)
- feature_fraction=0.07 (スイートスポット確定、I1/J2)
- learning_rate=0.02
- leaves=31/mcs=30はLB悪化 (訓練種過適合、L1/Q2)

## 6. 検証設計

### LOSO-CV (Leave-One-Species-Out)
- **必須**: train/testで樹種が完全非重複
- KFold CVは楽観的 (同一樹種がtrain/valに入る)
- LOSO-CVがLBと最も相関する唯一の指標

### アンサンブルの限界
- Seedアンサンブル: r=0.998で多様性なし (I4)
- LOSOバギング (13モデル平均): r=0.9963で効果なし (S6)
- I2×B2アンサンブル: r=0.9883でLB悪化 (J3)
→ 前処理パイプラインが同じ限り、多様性が生まれない

### ルール違反パターン
- テストデータのmean/stdを使用 → 禁止
- テスト測定順序 (delta_prev等) を使用 → 禁止
- 擬似ラベル (PL1/PL2/PL3) → 禁止
- 判断基準: 「未知のスペクトルが1つだけ与えられた状況でも実行可能か？」
